
# CSharp-DSA-VSCode-Template

A ready-to-use Visual Studio Code template for solving Data Structures and Algorithms (DSA) problems in C#. Includes input/output file handling, launch and task configurations for easy debugging, and a clean project structure to accelerate DSA practice and coding interviews.

## How to Use

1. Place your input in `input.txt`.
2. Press F5 or use the provided tasks to build and run the project.
3. Check `output.txt` for results.

## Project Structure
- Program.cs: Main logic, reads from input.txt and writes to output.txt
- input.txt: Input data
- output.txt: Output data
- .vscode/launch.json: Debug configuration
- .vscode/tasks.json: Build and run tasks

## Requirements
- .NET SDK (6.0 or later)
- Visual Studio Code
- C# extension for VS Code
